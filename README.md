# 📌 This Project

## 📖 Project Overview
This project is a Python console-based application designed to simulate the management of an open source software project.

It demonstrates how contributors, issues, and project data are handled using structured programming concepts. The system reflects real-world open source practices such as issue tracking, reporting, and collaboration.

---

## 🎯 Objectives
- Simulate an open source project environment
- Practice Python data structures and file handling
- Apply version control and project management concepts
- Understand issue tracking and contributor roles

---

## 🚀 Features
- Contributor registration and management
- Issue tracking system (Bug / Feature)
- Priority and status handling
- Data analysis (grouping, counting, sorting)
- File export (TXT and CSV reports)
- Urgent issue detection
